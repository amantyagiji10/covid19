https://api.covid19api.com/ - List all the current routes available with detail on each.

https://api.covid19api.com/summary - A summary of new and total cases per country updated daily.

https://api.covid19api.com/countries - Returns all the available countries and provinces, as well as the country slug for per country requests.

https://api.covid19api.com/dayone/country/south-africa/status/confirmed/live - Returns all cases by case type for a country from the first recorded case with the latest record being the live count. Country must be the Slug from /countries or /summary. Cases must be one of: confirmed, recovered, deaths

https://api.covid19api.com/country/south-africa/status/confirmed/live - get all the cases of country live from day one

https://api.covid19api.com/live/country/south-africa/status/confirmed/date/2020-03-21T13:13:30Z - Returns all live cases by case type for a country after a given date. These records are pulled every 10 minutes and are ungrouped. Country must be the slug from /countries or /summary. Cases must be one of: confirmed, recovered, deaths
